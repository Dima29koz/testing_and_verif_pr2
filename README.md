# testing_and_verif_pr2

Спецификация класса RailingCalc

Класс RailingCalc позволяет рассчитать положение балясин в пролете перил в зависимости от нескольких параметров:
•	Длина пролета (см)
•	Толщина балясины (см)
•	Расстояние между балясинами (см)
